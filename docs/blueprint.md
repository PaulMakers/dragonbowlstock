# **App Name**: DragonStock PWA

## Core Features:

- Unified Sheets Integration: Seamless real-time synchronization with the Google Apps Script backend to utilize Google Spreadsheets as the primary and only database.
- Multi-Date Stock Entry: Flexible inventory logging that allows employees to record stock depletion for current or historical dates, creating organized daily sheets automatically.
- Restock Taskmaster: A specialized 'Prepare Barang' workflow for supervisors to checklist items as they are restocked, updating statuses from 'Habis' to 'Sudah Tersedia' instantly.
- Visual Analytics Dashboard: Dynamic data visualization tracking item frequencies, 30-day stock trends, and current status distribution in a clean, modern interface.
- Admin Personnel Console: Comprehensive user management system to add and monitor employee accounts (including Candra, Novia, Indri, Lidia, Nanda, Arya) with secure authentication.
- Smart Export Suite: One-tap generation of PDF and Excel reports for daily reports or specific filtered results for physical printing and digital archiving.
- AI Demand Insights Tool: An AI-powered tool that analyzes historical usage patterns in the spreadsheet to reason and predict future stock requirements, highlighting critical items for restocking before they run out.

## Style Guidelines:

- Primary color: Radiant Ember Orange (#FF6B35) for a high-energy F&B brand identity that pops in low-light restaurant environments.
- Background: Obsidian Basalt (#1A1816), a warm dark-grey that complements the primary hue while reducing eye strain during night-shift use.
- Accent: Deep Sangria (#FF3B3B), used sparingly for alerts and 'Habis' status markers to ensure immediate visual priority.
- Headline font: 'Poppins', a geometric sans-serif that provides a contemporary, high-end digital agency feel; Body font: 'Inter', selected for its high legibility in complex data tables and mobile forms.
- Bold, 2px stroke-width icons in the style of Lucide, specifically designed for touch-targets of 44px or larger on mobile Android devices.
- Mobile-first card hierarchy with vertical rhythm, using thick borders and shadow depths to separate daily inventory sheets clearly.
- Slick layout transitions (Framer Motion style) when switching between date-sheets, and satisfying checkmark micro-interactions for the restock workflow.
